# Quick Review: Percentage 🔴

### Basic Percentage Conversion
**Formulas**:
Fraction to %: Multiply by 100
% to Fraction: Divide by 100
**Trick**: Memorize fraction equivalents: 1/2=50%, 1/3=33.33%, 1/4=25%, 1/5=20%, 1/6=16.66%, 1/7=14.28%, 1/8=12.5%, 1/9=11.11%, 1/11=9.09%.

---
### Percentage Increase/Decrease
**Formulas**:
Percentage change = (Difference / Initial Value) * 100
**Trick**: If A is x% more than B, then B is (x/(100+x))*100 % less than A.

---
### Successive Percentage Change
**Formulas**:
Net % change = x + y + (xy/100)
**Trick**: If both are increase, x and y are positive. If one is decrease, make it negative.

---
### Word Problems
**Formulas**:
Population after n years = P * (1 + R/100)^n
Marks: Pass mark = (Marks scored + Marks failed by) OR (Marks scored - Marks above passing)
**Trick**: Use 100 as initial value for easy calculation.
