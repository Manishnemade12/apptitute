# Percentage Formula Sheet (Aptitude)

> **For TCS, Infosys, Accenture, Capgemini, Cognizant, Wipro, Tech Mahindra and similar aptitude exams.**
>
> These are the core formulas required to solve the majority of percentage questions.

---

# 1. Basic Percentage

### Formula

`Percentage = (Part / Whole) × 100`

**Where**
- **Part** = Given value
- **Whole** = Total value

### Example

20 out of 80

`Percentage = (20 / 80) × 100 = 25%`

---

# 2. Find Part

### Formula

`Part = (Percentage × Whole) / 100`

### Example

20% of 500

`Part = (20 × 500) / 100 = 100`

---

# 3. Find Whole

### Formula

`Whole = (Part × 100) / Percentage`

### Example

40 is 20% of ?

`Whole = (40 × 100) / 20 = 200`

---

# 4. Percentage Increase

### Formula

`Increase % = [(New − Old) / Old] × 100`

### Example

100 → 125

`Increase = [(125 − 100) / 100] × 100 = 25%`

---

# 5. Percentage Decrease

### Formula

`Decrease % = [(Old − New) / Old] × 100`

### Example

200 → 150

`Decrease = [(200 − 150) / 200] × 100 = 25%`

---

# 6. New Value

### After Increase by x%

`New = Old × (100 + x) / 100`

### Example

₹500 increased by 20%

`New = 500 × 120 / 100 = ₹600`

### After Decrease by x%

`New = Old × (100 − x) / 100`

### Example

₹500 decreased by 20%

`New = 500 × 80 / 100 = ₹400`

---

# 7. Old Value

### If Increased by x%

`Old = New × 100 / (100 + x)`

### Example

New Value = ₹600 after 20% increase

`Old = 600 × 100 / 120 = ₹500`

### If Decreased by x%

`Old = New × 100 / (100 − x)`

### Example

New Value = ₹400 after 20% decrease

`Old = 400 × 100 / 80 = ₹500`

---

# 8. Successive Percentage Change

### Formula

`Net % = a + b ± (ab / 100)`

### Rules

- Increase + Increase → `+ab/100`
- Increase + Decrease → `−ab/100`
- Decrease + Decrease → Overall decrease

### Examples

**+20%, +10%**

`20 + 10 + (20×10)/100`

`= 20 + 10 + 2`

`= 32% Increase`

---

**+20%, −10%**

`20 − 10 − (20×10)/100`

`= 20 − 10 − 2`

`= 8% Increase`

---

**−20%, −10%**

`20 + 10 + (20×10)/100`

`= 32% Decrease`

---

# 9. Fixed Increase / Fixed Decrease

> Use only when increase/decrease is a **fixed amount**, **not a percentage**.

### Fixed Increase

`New = Old + (n × Increase)`

### Fixed Decrease

`New = Old − (n × Decrease)`

### Example

Salary = ₹50,000

Increase = ₹2,000 per year

After 5 years

`New = 50,000 + (5 × 2,000) = ₹60,000`

---

# 10. Expenditure (Price × Consumption)

### Formula

`Expenditure = Price × Consumption`

### If Price Increases by x%

Required Consumption Decrease

`Consumption Decrease % = [x / (100 + x)] × 100`

### Example

Price increases by 25%

`Consumption Decrease = (25 / 125) × 100 = 20%`

### If Price Decreases by x%

Required Consumption Increase

`Consumption Increase % = [x / (100 − x)] × 100`

### Example

Price decreases by 20%

`Consumption Increase = (20 / 80) × 100 = 25%`

# ⭐ Golden Formulas (Quick Revision)

1. `Percentage = (Part / Whole) × 100`

2. `Part = (Percentage × Whole) / 100`

3. `Whole = (Part × 100) / Percentage`

4. `Increase % = [(New − Old) / Old] × 100`

5. `Decrease % = [(Old − New) / Old] × 100`

6. `New = Old × (100 ± x) / 100`

7. `Old = New × 100 / (100 ± x)`

8. `Net % = a + b ± (ab / 100)`

9. `Expenditure = Price × Consumption`

10. `Consumption Change % = [x / (100 ± x)] × 100`
