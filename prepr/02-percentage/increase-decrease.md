# Percentage Increase & Decrease 🔴

## 📌 Core Concept
Percentage change measures how much a value has increased or decreased **relative to its original value**.

---

## 📝 All Formulas

### Percentage Change (General)
```
Percentage Change = [(New Value - Old Value) / Old Value] × 100
```
- If result is **positive** → Increase
- If result is **negative** → Decrease

### Percentage Increase
```
% Increase = [(Increase) / Original] × 100
New Value = Original × (1 + %/100)
           = Original × (100 + %) / 100
```

### Percentage Decrease
```
% Decrease = [(Decrease) / Original] × 100
New Value = Original × (1 - %/100)
           = Original × (100 - %) / 100
```

---

## 📝 Reverse Percentage (Critical Formula!)

### If A is x% MORE than B:
```
B is [x / (100 + x)] × 100 % LESS than A
```

### If A is x% LESS than B:
```
B is [x / (100 - x)] × 100 % MORE than A
```

### Multiplier Table (Memorize This!)

| % Change | Multiplier (Increase) | Multiplier (Decrease) |
|----------|----------------------|----------------------|
| 10% | × 11/10 | × 9/10 |
| 20% | × 6/5 | × 4/5 |
| 25% | × 5/4 | × 3/4 |
| 30% | × 13/10 | × 7/10 |
| 33.33% | × 4/3 | × 2/3 |
| 40% | × 7/5 | × 3/5 |
| 50% | × 3/2 | × 1/2 |
| 60% | × 8/5 | × 2/5 |
| 75% | × 7/4 | × 1/4 |
| 100% | × 2 | × 0 (zero!) |

---

## 📝 Expenditure / Consumption / Price Relationship
```
Expenditure = Price × Consumption
```

### If price increases by x%, to keep expenditure same:
```
% decrease in consumption = [x / (100 + x)] × 100
```

### If price decreases by x%, to keep expenditure same:
```
% increase in consumption = [x / (100 - x)] × 100
```

---

## 📝 Key Properties

| Property | Explanation |
|----------|-------------|
| Increase then same decrease ≠ original | 100 → +10% → 110 → -10% → 99 (NOT 100) |
| % change is always on the **original** | Never calculate on the new value unless specified |
| x% increase followed by x% decrease | Always results in a net **decrease** of x²/100 % |
| Doubling | = 100% increase |
| Tripling | = 200% increase |
| Halving | = 50% decrease |

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – Fraction Method**: Convert percentages to fractions for instant calculation.
> - If income is 25% more → income is 5/4 of the other
> - If income is 20% less → income is 4/5 of the other

> 💡 **Trick 2 – Reverse Percentage Shortcut**:
> - A is 25% more than B → B is 20% less than A (because 25/125 = 1/5 = 20%)
> - A is 20% less than B → B is 25% more than A (because 20/80 = 1/4 = 25%)
> - A is 50% more than B → B is 33.33% less than A

> 💡 **Trick 3 – Use 100 as Base**: For any abstract percentage problem, assume the original value is **100**. Makes calculation much simpler.

> 💡 **Trick 4 – Price × Consumption**: The formula for "keep expenditure same" is the reverse percentage formula. If price ↑ 20%, consumption ↓ by 20/120 × 100 = 16.66%.

## 📖 Solved Examples

### Example 1: Basic Increase
**Q**: A shirt costs ₹800. Its price increases by 15%. Find new price.  
**Solution**: New = 800 × (115/100) = 800 × 1.15 = **₹920**

---

### Example 2: Basic Decrease
**Q**: Population of a village was 25000. It decreased by 12%. Find new population.  
**Solution**: New = 25000 × (88/100) = **22000**

---

### Example 3: Reverse Percentage
**Q**: A's salary is 30% more than B's. By what % is B's salary less than A's?  
**Solution**:  
- Formula: x/(100+x) × 100
- = 30/130 × 100 = **23.07%**

---

### Example 4: Finding Original Value
**Q**: After a 20% increase, the price of a TV is ₹18000. What was the original price?  
**Solution**:  
- Original × (120/100) = 18000
- Original = 18000 × (100/120) = **₹15000**

---

### Example 5: Price-Consumption Problem
**Q**: If the price of sugar increases by 25%, by how much % must a family reduce consumption to keep expenditure the same?  
**Solution**:  
- Reduction = 25/(100+25) × 100
- = 25/125 × 100 = **20%**

---

### Example 6: Comparing Two Values
**Q**: If A's income is 40% less than B's income, then B's income is how much % more than A's?  
**Solution**:  
- Formula: x/(100-x) × 100
- = 40/60 × 100 = **66.66%**

---

### Example 7: Net Change After Increase & Decrease
**Q**: A value increases by 20% and then decreases by 20%. What is the net change?  
**Solution**:  
- Net change = -(20²/100) = -4% → **4% decrease**
- OR using 100: 100 → +20% → 120 → -20% → 96 → **4% decrease** ✅

---

## 🎯 Exam Tips
- The reverse percentage formula is asked in **almost every exam** — master it
- Always identify what is the "base" (original) for the percentage calculation
- Use the multiplier approach (fraction method) for speed
