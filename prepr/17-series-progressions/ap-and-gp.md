# AP and GP — Combined Reference 🟡/🟢

## AP Formulas
```
aₙ = a + (n-1)d    |    Sₙ = n/2 × [2a + (n-1)d]    |    Sₙ = n/2(a + l)
```

## GP Formulas
```
aₙ = arⁿ⁻¹    |    Sₙ = a(rⁿ-1)/(r-1)    |    S∞ = a/(1-r) [|r|<1]
```

## Quick Identities
- AP mean: b = (a+c)/2 | GP mean: b = √(ac)
- Sum of first n: n(n+1)/2
- Sum of squares: n(n+1)(2n+1)/6
- Sum of cubes: [n(n+1)/2]²
