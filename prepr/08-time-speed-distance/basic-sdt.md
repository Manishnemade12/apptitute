# Basic Speed, Distance & Time 🔴

## 📌 The Fundamental Relationship
```
Distance = Speed × Time
Speed = Distance / Time
Time = Distance / Speed
```

---

## 📝 Key Properties

| Property | Description |
|----------|-------------|
| D constant → S ∝ 1/T | Inverse proportion |
| S constant → D ∝ T | Direct proportion |
| T constant → D ∝ S | Direct proportion |
| If speed ratio = a:b | Time ratio = b:a (same distance) |
| If time ratio = a:b | Distance ratio = a:b (same speed) |

## 📝 Time Difference Problems

### Reaches Early/Late by Different Speeds
If a person travels at speed S₁ and reaches t₁ late, at speed S₂ reaches t₂ early:
```
Distance = S₁ × S₂ × (t₁ + t₂) / (S₂ - S₁)
```
(Convert t₁, t₂ to hours if given in minutes)

### Same Distance, Different Speeds
```
D/S₁ - D/S₂ = Time difference
D = (S₁ × S₂ × ΔT) / (S₂ - S₁)    if S₂ > S₁
```

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – 5/18 and 18/5**: Fastest conversion. 72 kmph = 72 × 5/18 = 20 m/s.

> 💡 **Trick 2 – Ratio Method**: If speeds are 3:4 (same distance), times are 4:3. If time difference = x, find each time using the ratio difference.

> 💡 **Trick 3 – Distance = Speed × Time**: When stuck, always come back to this. Draw a diagram if needed.

## 📖 Solved Examples

### Example 1: Basic
**Q**: A car travels 240 km in 4 hours. Speed?  
**Solution**: 240/4 = **60 kmph**

---

### Example 2: Unit Conversion
**Q**: Express 90 kmph in m/s.  
**Solution**: 90 × 5/18 = **25 m/s**

---

### Example 3: Time Difference
**Q**: A person walks at 4 kmph and reaches 10 min late. If he walks at 5 kmph, he reaches 10 min early. Find distance.  
**Solution**:  
- D = (4×5×(10+10))/(5-4) × 1/60 [convert min to hrs]
- D = (20×20/60)/1 = 400/60 = **6.67 km**
- Or: D/4 - D/5 = 20/60 → D(5-4)/20 = 1/3 → D/20 = 1/3 → D = **20/3 = 6.67 km**

---

### Example 4: Speed Ratio
**Q**: Two cars travel the same distance. Speed ratio 3:5. If slower car takes 40 min more, find the time taken by each.  
**Solution**:  
- Time ratio = 5:3 (inverse of speed)
- Difference = 5x - 3x = 2x = 40 min → x = 20
- Slower = 100 min, Faster = **60 min**
