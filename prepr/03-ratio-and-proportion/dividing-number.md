# Dividing a Number in a Given Ratio 🔴

## 📌 Concept
When a total quantity is divided into parts according to a given ratio, each part can be calculated using a simple formula.

---

## 📝 All Formulas

### Two-Part Division
If a total **S** is divided in ratio **a : b**:
```
First part  = S × a / (a + b)
Second part = S × b / (a + b)
```

### Three-Part Division
If total **S** is divided in ratio **a : b : c**:
```
First part  = S × a / (a + b + c)
Second part = S × b / (a + b + c)
Third part  = S × c / (a + b + c)
```

### General n-Part Division
If S is divided in ratio a₁ : a₂ : ... : aₙ:
```
k-th part = S × aₖ / (a₁ + a₂ + ... + aₙ)
```

---

## 📝 Key Properties

| Property | Explanation |
|----------|-------------|
| Sum of parts = Total | Always verify your answer by adding all parts |
| Ratio of parts is preserved | The parts will be in the same ratio as given |
| Value of one "unit" = S / (sum of ratio terms) | This is the fundamental building block |

---

## 📝 Common Variations

### Finding the Difference Between Parts
If S is divided in ratio a:b (a > b):
```
Difference = S × (a - b) / (a + b)
```

### Given Difference, Find Parts
If the difference between two parts is D and ratio is a:b:
```
S = D × (a + b) / (a - b)
```

### Changing the Ratio
If a sum was first divided as a:b and then changed to p:q:
- Calculate parts in both ratios
- The change for each person = New part - Old part

---

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – Unit Value Method**: Find 1 unit = Total ÷ Sum of ratio terms. Then multiply by each term.
> - Divide 350 in 3:4. One unit = 350/7 = 50. Parts = 150 and 200.

> 💡 **Trick 2 – Difference Shortcut**: For 2-part division, if you know the difference and ratio:
> - S = Difference × (sum of terms) / (difference of terms)

> 💡 **Trick 3 – Combining Division**: Sometimes a ratio changes with conditions (like age after years). Set up equations using the ratio formula.

---

## 📖 Solved Examples

### Example 1: Basic Two-Part
**Q**: Divide 1050 in the ratio 2:5.  
**Solution**:  
- Sum = 7, One unit = 150
- Parts = 2×150 = **300** and 5×150 = **750**
- Verify: 300 + 750 = 1050 ✅

---

### Example 2: Three-Part Division
**Q**: ₹2400 is divided among A, B, C in ratio 3:4:5. Find each share.  
**Solution**:  
- Sum = 12, One unit = 200
- A = **600**, B = **800**, C = **1000**

---

### Example 3: Given Difference
**Q**: Two numbers are in ratio 7:3 and their difference is 120. Find the numbers.  
**Solution**:  
- Difference of terms = 7-3 = 4
- One unit = 120/4 = 30
- Numbers = 7×30 = **210** and 3×30 = **90**

---

### Example 4: Age Problem
**Q**: Ages of A and B are in ratio 5:3. After 6 years, ratio becomes 7:5. Find present ages.  
**Solution**:  
- Let ages = 5x and 3x
- (5x + 6)/(3x + 6) = 7/5
- 25x + 30 = 21x + 42
- 4x = 12, x = 3
- Ages: **15 years** and **9 years**

---

### Example 5: Salary Distribution
**Q**: A, B, C share profits in ratio 5:4:3. If C's share is ₹1200, find total profit.  
**Solution**:  
- C = 3 units = 1200 → 1 unit = 400
- Total = 12 units = **₹4800**

---

### Example 6: Changing Ratio
**Q**: ₹3600 is divided among A, B, C in ratio 3:5:4. If ₹400 is taken from B and given to A, what is the new ratio?  
**Solution**:  
- Original: A=900, B=1500, C=1200
- New: A=1300, B=1100, C=1200
- New ratio = **13:11:12**

---

## 🎯 Exam Tips
- Always find the "one unit" value first — everything else follows
- Verify your answer by checking sum = total
- Age problems + ratio is a very common combo — set up equations methodically
