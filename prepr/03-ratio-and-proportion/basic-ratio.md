## рџ“ќ Combining Ratios

### When one term is common:
If **A:B = m:n** and **B:C = p:q**, to find **A:B:C**:
1. Make B the same in both ratios
2. B = LCM(n, p) or simply cross-multiply:
```
A : B : C = (mГ—p) : (nГ—p) : (nГ—q)
```

**Example**: A:B = 2:3, B:C = 4:5

### When two terms overlap:
