# Number of Factors

## Formulas
If N = p^a * q^b * r^c..., then Number of factors = (a+1)(b+1)(c+1)...

## Tricks & Shortcuts
> 💡 To find sum of factors: [ (p^(a+1)-1)/(p-1) ] * [ (q^(b+1)-1)/(q-1) ] ...

## Examples
**Q**: Number of factors of 100?
**Ans**: 100 = 2^2 * 5^2. Factors = (2+1)*(2+1) = 9.
