# Marked Price Problems 🟡

## 📌 What is Marked Price?
**Marked Price (MP)** = the price "marked" or labeled on the product (also called List Price or MRP).

The **relationship chain** is: **CP → MP → SP**
- Shopkeeper marks up CP to get MP
- Then gives discount on MP to get SP

---

## 📝 All Formulas

### Core Relationships
```
MP = CP + Markup
SP = MP - Discount
Profit = SP - CP = (MP - Discount) - CP
```

### Markup Percentage (on CP)
```
Markup % = [(MP - CP) / CP] × 100
MP = CP × (100 + Markup%) / 100
```

### Discount Percentage (on MP)
```
Discount % = [(MP - SP) / MP] × 100
SP = MP × (100 - Discount%) / 100
```

### The Golden Ratio Formula
```
MP / CP = (100 + Profit%) / (100 - Discount%)
```
This single formula connects all four quantities!

### If Profit%, Discount% given, Find Markup%
```
(100 + Markup%) = [(100 + Profit%) / (100 - Discount%)] × 100
```

---

## 📝 Key Properties

| Property | Description |
|----------|-------------|
| Markup is on CP | Shopkeeper adds markup on his cost |
| Discount is on MP | Customer gets discount on the label price |
| Profit is on CP | Always calculated on cost price |
| If Discount% = Markup% | There's always a **loss** |
| No discount → SP = MP | Profit = Markup |

---

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – The Golden Ratio**: MP/CP = (100+P%)/(100-D%). This solves most problems in one step.
> - 20% profit, 10% discount: MP/CP = 120/90 = 4/3 → Markup = 33.33%

> 💡 **Trick 2 – Assume CP = 100**: Makes all calculations instant.
> - CP=100, 30% markup → MP=130, 10% discount → SP=130×0.9=117 → 17% profit

> 💡 **Trick 3 – Successive Changes**: Think of the problem as:
> - CP → (+Markup%) → MP → (-Discount%) → SP
> - Use: Net % change = M - D - (MD/100) where M=markup%, D=discount%
> - If result is +ve → Profit, if -ve → Loss

## 📖 Solved Examples

### Example 1: Basic MP Problem
**Q**: A shopkeeper marks an article 40% above CP and gives 10% discount. Find profit%.  
**Solution**:  
- Let CP = 100. MP = 140. SP = 140 × 90/100 = 126
- Profit = 26%. 
- Or: Net = 40 - 10 - (400/100) = 40 - 10 - 4 = **26%** ✅

---

### Example 2: Finding MP
**Q**: CP = ₹300, Profit% = 20%, Discount% = 10%. Find MP.  
**Solution**:  
- MP/CP = (100+20)/(100-10) = 120/90 = 4/3
- MP = 300 × 4/3 = **₹400**
- Verify: SP = 400 × 0.9 = 360, Profit = 60/300 = 20% ✅

---

### Example 3: Finding Discount%
**Q**: A shopkeeper marks goods 50% above CP. What discount % should he give to earn 20% profit?  
**Solution**:  
- MP/CP = 3/2, SP/CP = 6/5
- SP/MP = (6/5)/(3/2) = 12/15 = 4/5
- SP = 4/5 × MP → Discount = 1/5 = **20%**

---

### Example 4: Finding CP
**Q**: MP = ₹840, Discount = 10%, Profit = 12%. Find CP.  
**Solution**:  
- SP = 840 × 90/100 = 756
- CP = 756 × 100/112 = **₹675**

---

### Example 5: Loss Scenario
**Q**: An item marked at ₹500 is sold at 30% discount. If CP is ₹400, find profit or loss%.  
**Solution**:  
- SP = 500 × 70/100 = 350
- Loss = 400 - 350 = 50
- Loss% = 50/400 × 100 = **12.5% loss**

---

### Example 6: Two Successive Discounts + Profit
**Q**: MP is 60% above CP. If two discounts of 15% and 10% are given, find profit%.  
**Solution**:  
- Let CP = 100, MP = 160
- SP = 160 × 85/100 × 90/100 = 160 × 0.765 = 122.4
- Profit = **22.4%**

---

## 🎯 Exam Tips
- The golden ratio formula (MP/CP = ...) is the single most useful formula
- "Assume CP = 100" makes almost every problem trivial
- Markup ≠ Profit unless there's no discount
