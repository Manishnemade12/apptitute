# Dishonest Dealer / False Weight Problems 🟢

## 📌 Concept
A **dishonest dealer** cheats by using **false weights** (giving less quantity than what the customer pays for) while charging at cost price or with additional markup.

---

## 📝 All Formulas

### Basic Dishonest Dealer (Sells at CP, Uses False Weight)
```
Profit % = (Error / True Value - Error) × 100
         = (True Weight - False Weight) / False Weight × 100
```

### Alternative Form
```
Profit % = [(Claimed Weight / Actual Weight) - 1] × 100
```

### Dishonest Dealer + Additional Profit/Loss
If dealer also charges x% above/below CP:
```
Total Gain % = [(100 + x) × (True Weight / False Weight)] - 100 %
```
Where x is +ve for selling above CP, -ve for selling below CP.

### General Formula (Using Ratio)
```
Profit % = [(Claims to sell) / (Actually sells) × (SP/CP) - 1] × 100
```

---

## 📝 Key Properties

| Property | Description |
|----------|-------------|
| Profit is on actual goods sold | He pays for less, receives money for more |
| Selling at CP doesn't mean 0 profit | If weights are false, he still profits |
| Selling below CP can still be profitable | If the weight error compensates |

---

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – Think in Ratios**: If he claims 1 kg but gives 800g:
> - He sells 800g worth at 1000g price
> - Profit % = (200/800) × 100 = 25%

> 💡 **Trick 2 – Error/True or Error/False**: 
> - Profit% = Error/Actual given × 100 (Error = what he keeps)
> - Error = True value - False value = 1000 - 800 = 200

> 💡 **Trick 3 – Combined**: If also selling at x% profit with false weight:
> - Total = (100+x)/100 × True/False × 100 - 100 %

## 📖 Solved Examples

### Example 1: Basic False Weight
**Q**: A dealer professes to sell at CP but uses 900g weight instead of 1 kg. Find his profit%.  
**Solution**:  
- Error = 1000 - 900 = 100g
- Profit% = (100/900) × 100 = **11.11%**

---

### Example 2: Different False Weight
**Q**: A dishonest shopkeeper sells at CP but uses 800g weight for 1 kg. Profit%?  
**Solution**: Profit% = (200/800) × 100 = **25%**

---

### Example 3: False Weight + Markup
**Q**: A dealer sells goods at 10% above CP and uses 900g for 1 kg. Total gain%?  
**Solution**:  
- Gain = [(110/100) × (1000/900) - 1] × 100
- = [1.1 × 10/9 - 1] × 100
- = [1.222 - 1] × 100 = **22.22%**

---

### Example 4: Selling Below CP
**Q**: A dealer claims to sell at 4% loss but uses 800g weight for 1 kg. Actual profit/loss%?  
**Solution**:  
- Gain = [(96/100) × (1000/800) - 1] × 100
- = [0.96 × 1.25 - 1] × 100
- = [1.2 - 1] × 100 = **20% profit**

---

### Example 5: Finding False Weight
**Q**: A dealer makes 25% profit by using false weights while selling at CP. What weight does he use for 1 kg?  
**Solution**:  
- 25 = (1000-w)/w × 100
- 0.25w = 1000-w → 1.25w = 1000 → w = **800g**

---

## 🎯 Exam Tips
- Remember: Profit% = Error/Actual × 100 (not Error/True)
- These are typically easy 🟢 priority questions — don't skip them
- If selling below CP but still making profit, the weight fraud must be significant
