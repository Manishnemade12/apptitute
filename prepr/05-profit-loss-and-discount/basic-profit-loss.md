# Basic Profit/Loss % 🔴

## 📌 Definitions
- **Cost Price (CP)**: The price at which an article is **bought/purchased**
- **Selling Price (SP)**: The price at which an article is **sold**
- **Profit**: When SP > CP → Profit = SP - CP
- **Loss**: When CP > SP → Loss = CP - SP

---

## 📝 All Formulas

### Profit & Loss Amount
```
Profit = SP - CP
Loss   = CP - SP
```

### Profit & Loss Percentage (ALWAYS on CP!)
```
Profit % = (Profit / CP) × 100 = [(SP - CP) / CP] × 100
Loss %   = (Loss / CP) × 100   = [(CP - SP) / CP] × 100
```

### Finding SP from CP
```
SP = CP × (100 + Profit%) / 100    ← Profit case
SP = CP × (100 - Loss%) / 100      ← Loss case
```

### Finding CP from SP
```
CP = SP × 100 / (100 + Profit%)    ← Profit case
CP = SP × 100 / (100 - Loss%)      ← Loss case
```

### Multiplier Approach (Fastest)

| Profit/Loss | Multiplier (SP = CP ×) |
|-------------|------------------------|
| 10% profit | × 11/10 |
| 20% profit | × 6/5 |
| 25% profit | × 5/4 |
| 33.33% profit | × 4/3 |
| 50% profit | × 3/2 |
| 10% loss | × 9/10 |
| 20% loss | × 4/5 |
| 25% loss | × 3/4 |

---

## 📝 Key Properties

| Property | Description |
|----------|-------------|
| % is ALWAYS on CP | Unless specifically stated otherwise |
| If SP₁ = SP₂, one at x% profit & one at x% loss | **Always a net loss** of x²/100 % |
| If SP of x articles = CP of y articles | Profit % = [(y-x)/x] × 100 |
| Buy x, get y free | Loss on each = y/(x+y) × 100 % → Actually this is profit for buyer, loss for seller: Profit% = y/x × 100 conceptually — but seller's profit% = [y/(x+y)] if pricing differently |

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – CP of x = SP of y**: This is the most common shortcut.
> - CP of 15 = SP of 12 → Profit % = (15-12)/12 × 100 = 25%
> - CP of 10 = SP of 12 → Loss % = (12-10)/12 × 100... **NO!** 
> - Correct: If CP of x items = SP of y items and y < x → Profit, y > x → Loss
> - Profit/Loss % = |x-y|/y × 100 (when expressed as SP of y = CP of x)
> - **Actually**: If SP of x = CP of y → profit% = (y-x)/x × 100

> 💡 **Trick 2 – Same SP, Same % Profit & Loss**: Always net **loss** = x²/100 %.
> - 20% profit and 20% loss on same SP → Net loss = 4%

> 💡 **Trick 3 – Use the Multiplier**: Don't calculate profit/loss first. Just multiply CP by the fraction.
> - 25% profit on ₹600 → SP = 600 × 5/4 = ₹750

## 📖 Solved Examples

### Example 1: Basic Profit
**Q**: An article bought for ₹400 is sold for ₹500. Find profit %.  
**Solution**: Profit = 100. Profit% = (100/400) × 100 = **25%**

---

### Example 2: Basic Loss
**Q**: A book bought for ₹250 is sold for ₹200. Find loss %.  
**Solution**: Loss = 50. Loss% = (50/250) × 100 = **20%**

---

### Example 3: Finding SP
**Q**: CP = ₹1500, Profit = 20%. Find SP.  
**Solution**: SP = 1500 × 120/100 = **₹1800**

---

### Example 4: Finding CP
**Q**: SP = ₹720, Loss = 10%. Find CP.  
**Solution**: CP = 720 × 100/90 = **₹800**

---

### Example 5: SP of x = CP of y
**Q**: The CP of 20 articles is equal to the SP of 16 articles. Find profit/loss %.  
**Solution**:  
- Let CP per article = ₹1, so CP of 20 = ₹20
- SP of 16 = ₹20, so SP per article = 20/16 = 5/4
- SP > CP → Profit
- Profit % = [(5/4 - 1)/1] × 100 = **25%**

---

### Example 6: Same SP, Same % P&L
**Q**: Two articles sold at ₹990 each. One at 10% profit, other at 10% loss. Find overall profit/loss.  
**Solution**:  
- Net loss% = 10²/100 = **1% loss**
- CP₁ = 990 × 100/110 = 900
- CP₂ = 990 × 100/90 = 1100
- Total CP = 2000, Total SP = 1980
- Loss = 20, Loss% = 20/2000 × 100 = **1% loss** ✅

---

## 🎯 Exam Tips
- Always identify CP before calculating percentage
- The "CP of x = SP of y" pattern appears in almost every exam
- Same %, one profit one loss → always net loss → great for MCQ elimination
