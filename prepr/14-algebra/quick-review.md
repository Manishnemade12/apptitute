# ⚡ Quick Review: Algebra 🟢

---

## Linear Equations
- ax + b = 0 → x = -b/a
- Two variables: Substitution or Elimination
- Unique solution: a₁/a₂ ≠ b₁/b₂

## Quadratic: ax² + bx + c = 0
```
x = [-b ± √(b²-4ac)] / 2a
Sum of roots = -b/a | Product = c/a
```
D > 0: two real | D = 0: equal | D < 0: no real

## Must-Know Identities
- (a±b)² = a² ± 2ab + b² | a²-b² = (a+b)(a-b)
- a³±b³ = (a±b)(a²∓ab+b²)
