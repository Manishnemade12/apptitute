# Quadratic Equations 🟢

## 📌 Standard Form
```
ax² + bx + c = 0
```

### Roots (Solutions)
```
x = [-b ± √(b²-4ac)] / 2a
```

### Discriminant (D = b²-4ac)
| D Value | Nature of Roots |
|---------|----------------|
| D > 0 | Two distinct real roots |
| D = 0 | Two equal real roots |
| D < 0 | No real roots (complex) |

### Sum and Product of Roots
If α and β are roots:
```
α + β = -b/a
α × β = c/a
```

## 📝 Algebraic Identities (Must Know)
```
(a+b)² = a² + 2ab + b²
(a-b)² = a² - 2ab + b²
a² - b² = (a+b)(a-b)
(a+b)³ = a³ + 3a²b + 3ab² + b³
(a-b)³ = a³ - 3a²b + 3ab² - b³
a³ + b³ = (a+b)(a²-ab+b²)
a³ - b³ = (a-b)(a²+ab+b²)
(a+b+c)² = a² + b² + c² + 2(ab+bc+ca)
```

## 📖 Solved Examples

### Example 1
**Q**: x² - 5x + 6 = 0. Find roots.  
**Solution**: (x-2)(x-3)=0 → x = **2, 3**. Sum=5, Product=6 ✅

### Example 2
**Q**: Sum of roots = 7, product = 12. Find the equation.  
**Solution**: x² - 7x + 12 = 0. Roots: **3 and 4**
