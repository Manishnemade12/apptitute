# Logarithm — Basic Rules 🟢

## 📌 Definition
```
If aˣ = N, then log_a(N) = x
```
"log base a of N equals x"

---

## 📝 All Log Rules

### Basic Properties
| Rule | Formula |
|------|---------|
| **Product Rule** | log(AB) = log A + log B |
| **Quotient Rule** | log(A/B) = log A - log B |
| **Power Rule** | log(Aⁿ) = n × log A |
| **Change of Base** | log_a(B) = log B / log A |
| **log 1** | log_a(1) = 0 (for any base) |
| **log of base** | log_a(a) = 1 |
| **Reciprocal** | log_a(b) = 1/log_b(a) |
| **Chain Rule** | log_a(b) × log_b(c) = log_a(c) |

### Common Logarithm Values (Base 10)
```
log 1 = 0       log 10 = 1      log 100 = 2
log 2 ≈ 0.301   log 3 ≈ 0.477   log 5 ≈ 0.699
log 7 ≈ 0.845
```

### Natural Logarithm (ln, base e ≈ 2.718)
```
ln 1 = 0    ln e = 1    ln e² = 2
```

---

## 📝 Key Properties
| Property | Description |
|----------|-------------|
| log is only defined for positive numbers | log(0) and log(negative) are undefined |
| log_a(x) increases with x (if a>1) | Monotonically increasing |
| log_a(x) < 0 when 0 < x < 1 | For a > 1 |

## 📖 Solved Examples

### Example 1
**Q**: log₂(32) = ?  
**Solution**: 2⁵ = 32, so log₂(32) = **5**

### Example 2
**Q**: log(25) + log(4) = ?  
**Solution**: log(25×4) = log(100) = **2**

### Example 3
**Q**: If log x = 2.5, find x (base 10).  
**Solution**: x = 10^2.5 = 10² × 10^0.5 = 100√10 ≈ **316.23**

### Example 4
**Q**: log₃(81) × log₈₁(9) = ?  
**Solution**: log₃(81) = 4. log₈₁(9) = log₃(9)/log₃(81) = 2/4 = 1/2. Product = 4 × 1/2 = **2**
Or chain rule: log₃(81) × log₈₁(9) = log₃(9) = **2** ✅
