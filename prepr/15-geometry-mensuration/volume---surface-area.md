# Volume & Surface Area 🟡

## 📝 All Formulas

### Cube (side = a)
```
Volume = a³
TSA = 6a²     |    LSA = 4a²
Diagonal = a√3
```

### Cuboid (l × b × h)
```
Volume = l × b × h
TSA = 2(lb + bh + lh)    |    LSA = 2h(l + b)
Diagonal = √(l² + b² + h²)
```

### Cylinder (radius r, height h)
```
Volume = πr²h
TSA = 2πr(r + h)    |    CSA = 2πrh
```

### Cone (radius r, height h, slant l)
```
Volume = (1/3)πr²h
TSA = πr(l + r)    |    CSA = πrl
Slant height l = √(r² + h²)
```

### Sphere (radius r)
```
Volume = (4/3)πr³
Surface Area = 4πr²
```

### Hemisphere (radius r)
```
Volume = (2/3)πr³
TSA = 3πr²    |    CSA = 2πr²
```

---

## 📝 Key Relationships
| Relation | Formula |
|----------|---------|
| Cone:Cylinder:Sphere (same r, h=2r) | Volume ratio = **1:3:2** |
| Sphere in cylinder | Sphere volume = 2/3 × Cylinder volume |

## 📖 Solved Examples

### Example 1
**Q**: Cylinder r=7, h=10. Volume?  
**Solution**: π×49×10 = 22/7×490 = **1540 cm³**

### Example 2
**Q**: Sphere r=3. Volume?  
**Solution**: 4/3 × π × 27 = **36π = 113.1 cm³**

## Quick Review
- Cube: V=a³, SA=6a² | Cuboid: V=lbh, SA=2(lb+bh+lh)
- Cylinder: V=πr²h | Cone: V=(1/3)πr²h | Sphere: V=(4/3)πr³
- Volume ratio (cone:cyl:sphere) = 1:3:2
