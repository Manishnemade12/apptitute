# Area & Perimeter (Basic Shapes) 🟡

## 📝 All Formulas

### Rectangle
```
Area = l × b    |    Perimeter = 2(l + b)    |    Diagonal = √(l² + b²)
```

### Square
```
Area = a²    |    Perimeter = 4a    |    Diagonal = a√2
```

### Triangle
```
Area = ½ × base × height
Area (Heron's) = √[s(s-a)(s-b)(s-c)]    where s = (a+b+c)/2
Equilateral: Area = (√3/4) × a²,  Height = (√3/2) × a
```

### Circle
```
Area = πr²    |    Circumference = 2πr    |    Diameter = 2r
Semicircle Area = πr²/2    |    Semicircle Perimeter = πr + 2r
```

### Parallelogram
```
Area = base × height
```

### Trapezium
```
Area = ½ × (sum of parallel sides) × height = ½(a+b)×h
```

### Rhombus
```
Area = ½ × d₁ × d₂    (product of diagonals / 2)
Perimeter = 4a
Side = ½√(d₁² + d₂²)
```

---

## 📝 Key Properties
- Rectangle diagonal creates two equal right triangles
- Square diagonal = side × √2
- In equilateral triangle, all heights are equal = (√3/4)a
- Quadrilateral inscribed in circle: sum of opposite angles = 180°

## 📖 Solved Examples

### Example 1
**Q**: Rectangle 12×8. Area and diagonal?  
**Solution**: Area = 96, Diagonal = √(144+64) = √208 = **4√13 ≈ 14.42**

### Example 2
**Q**: Equilateral triangle side 10cm. Area?  
**Solution**: (√3/4) × 100 = **25√3 ≈ 43.3 cm²**

### Example 3
**Q**: Circle radius 7cm. Area?  
**Solution**: π × 49 = **154 cm²** (using π = 22/7)

---

## 🎯 Exam Tips
- Use π = 22/7 or 3.14 based on what simplifies the calculation
- For Heron's formula, find s first
- Area of rhombus using diagonals is the easiest method
