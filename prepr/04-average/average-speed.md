# Average Speed Problems

## Formulas
Average Speed = Total Distance / Total Time
If distance is same (A to B and B to A with speeds x and y): Avg Speed = 2xy / (x+y)

## Tricks & Shortcuts
> 💡 Never average the speeds directly if time taken is different.

## Examples
**Q**: Go at 40 kmph, return at 60 kmph.
**Ans**: Avg speed = 2*40*60 / (40+60) = 4800/100 = 48 kmph.
