# Simple Average 🔴

## 📌 What is Average?
Average (or Arithmetic Mean) is the **central value** of a set of numbers, calculated by dividing the **sum** of all observations by the **number** of observations.

---

## 📝 Core Formula
```
Average = Sum of all observations / Number of observations
Sum = Average × Number of observations
Number of observations = Sum / Average
```

---

## 📝 Important Formulas for Sequences

### Average of First n Natural Numbers (1, 2, 3, ..., n)
```
Average = (n + 1) / 2
```

### Average of First n Even Numbers (2, 4, 6, ..., 2n)
```
Average = (n + 1)
```

### Average of First n Odd Numbers (1, 3, 5, ..., 2n-1)
```
Average = n
```

### Average of Consecutive Numbers from a to b
```
Average = (a + b) / 2
```

### Sum of First n Natural Numbers
```
Sum = n(n+1)/2
```

### Sum of Squares of First n Natural Numbers
```
Sum = n(n+1)(2n+1)/6
```

### Sum of Cubes of First n Natural Numbers
```
Sum = [n(n+1)/2]²
```

---

## 📝 Key Properties

| Property | Description |
|----------|-------------|
| If every observation increased by k | New average = Old average + k |
| If every observation decreased by k | New average = Old average - k |
| If every observation multiplied by k | New average = Old average × k |
| If every observation divided by k | New average = Old average / k |
| Average of an AP | = (First term + Last term) / 2 |
| Average always lies between min and max | min ≤ Average ≤ max |

---

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – Deviation Method**: Choose an assumed mean (a), find deviations (d = obs - a), then:
> Average = a + (Sum of deviations / n)
> - Great when numbers are close together. E.g., 48, 51, 50, 47, 54 → assume 50 → deviations: -2, 1, 0, -3, 4 → sum = 0 → avg = 50.

> 💡 **Trick 2 – For AP (Arithmetic Progression)**: Average = (First + Last)/2. No need to add all terms.

> 💡 **Trick 3 – Consecutive Numbers**: Average of any set of consecutive numbers = middle number (if odd count) or average of two middle numbers (if even count).

> 💡 **Trick 4 – Quick Sum**: Sum = Average × Count. If you know the average, you can find the total instantly.

## 📖 Solved Examples

### Example 1: Basic Average
**Q**: Find the average of 12, 18, 22, 14, 24.  
**Solution**: Sum = 90, Count = 5. Average = 90/5 = **18**

---

### Example 2: First n Natural Numbers
**Q**: Average of first 50 natural numbers?  
**Solution**: (50+1)/2 = **25.5**

---

### Example 3: Consecutive Numbers
**Q**: Average of all integers from 10 to 40?  
**Solution**: (10+40)/2 = **25**

---

### Example 4: Using Property
**Q**: Average of 5 numbers is 20. If each number is multiplied by 3, what is the new average?  
**Solution**: New average = 20 × 3 = **60**

---

### Example 5: Finding Missing Number
**Q**: Average of 6 numbers is 30. Five of them are 25, 35, 28, 32, 40. Find the 6th.  
**Solution**:  
- Sum = 30 × 6 = 180
- Sum of 5 = 160
- 6th number = 180 - 160 = **20**

---

### Example 6: Deviation Method
**Q**: Find average of 198, 202, 195, 205, 200.  
**Solution**:  
- Assume mean = 200
- Deviations: -2, +2, -5, +5, 0 → Sum = 0
- Average = 200 + 0/5 = **200**

---

## 🎯 Exam Tips
- Remember the AP average formula — it works for any evenly spaced set
- The deviation method is the fastest for mental calculation of clustered data
- Sum = Average × Count — use this relationship constantly
