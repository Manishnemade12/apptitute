# ⚡ Quick Review: Time & Work 🔴

---

## Core Concept
```
Work = Efficiency × Time
LCM Method: Total Work = LCM of all given days
Efficiency = Total Work / Individual days
```

## Combined Work
| Scenario | Formula |
|----------|---------|
| A + B together | (A×B)/(A+B) days |
| A + B + C together | 1/T = 1/A + 1/B + 1/C |
| Finding individual from pairs | 2×(A+B+C) = (A+B) + (B+C) + (A+C) |
| Alternate days | 2-day output, then divide |

## Pipes & Cisterns
- Inlet = **+ve** efficiency, Outlet/Leak = **-ve** efficiency
- Same LCM method
- Leak: empties in = **(a×b)/(b-a)** hrs (pipe alone=a, with leak=b)

## Work Equivalence (MDH Rule)
```
M₁ × D₁ × H₁ / W₁ = M₂ × D₂ × H₂ / W₂
```
- Convert different worker types to one unit first
- Man-days = constant for same work

## Key Relationships
- Efficiency ∝ 1/Time | More workers → Less time
- Eff ratio of A:B = Time ratio B:A (inverse)
