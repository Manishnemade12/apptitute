# ⚡ Quick Review: Ages 🟡

---

## Core Rules
- Age after n years = Present + n | Age n years ago = Present - n
- **Age difference is ALWAYS constant**
- Ratio of ages **changes** with time

## Problem Types
| Type | Method |
|------|--------|
| Present ratio + Future ratio | Let ages = ax, bx. Set (ax+n)/(bx+n) = c/d |
| Present ratio + Past ratio | Let ages = ax, bx. Set (ax-n)/(bx-n) = c/d |
| Ratio + Difference | Diff = (a-b)x → find x |
| Ratio + Sum | Sum = (a+b)x → find x |
| "Twice as old" | A = 2B, set up equation |

## Quick Trick
- In MCQs, plug in options — fastest for age problems
