# Surds & Indices 🟡

## 📌 Indices (Exponents/Powers)

### All Laws of Indices
| Law | Formula |
|-----|---------|
| Product | aᵐ × aⁿ = aᵐ⁺ⁿ |
| Quotient | aᵐ ÷ aⁿ = aᵐ⁻ⁿ |
| Power of power | (aᵐ)ⁿ = aᵐⁿ |
| Zero exponent | a⁰ = 1 |
| Negative exponent | a⁻ⁿ = 1/aⁿ |
| Fractional exponent | a^(1/n) = ⁿ√a |
| Product of same power | aⁿ × bⁿ = (ab)ⁿ |
| Quotient of same power | aⁿ ÷ bⁿ = (a/b)ⁿ |

## 📌 Surds

### Definition
A surd is an irrational root: √2, ∛5, etc.

### Operations
```
√a × √b = √(ab)
√a / √b = √(a/b)
(√a)² = a
a√b + c√b = (a+c)√b
```

### Rationalization
```
1/(√a + √b) = (√a - √b) / (a - b)    [multiply by conjugate]
1/(√a - √b) = (√a + √b) / (a - b)
```

### Important Square Roots
| √ | Value |
|---|-------|
| √2 | 1.414 |
| √3 | 1.732 |
| √5 | 2.236 |
| √7 | 2.646 |

## 📖 Solved Examples

### Example 1: Indices
**Q**: Simplify 2⁵ × 2³ ÷ 2⁴  
**Solution**: 2^(5+3-4) = 2⁴ = **16**

---

### Example 2: Rationalize
**Q**: 1/(√3 + √2) = ?  
**Solution**: (√3 - √2)/(3-2) = **√3 - √2**

---

### Example 3: Compare Surds
**Q**: Which is greater: ∛4 or √3?  
**Solution**: ∛4 = 4^(1/3) = 4^(2/6) = 16^(1/6); √3 = 3^(1/2) = 3^(3/6) = 27^(1/6). Since 27 > 16, **√3 > ∛4**

## Quick Review
- Laws of indices: add/subtract powers for multiply/divide
- Surds: rationalize by multiplying with conjugate
- √2 ≈ 1.414, √3 ≈ 1.732, √5 ≈ 2.236
