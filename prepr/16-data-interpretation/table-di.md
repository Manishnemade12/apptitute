# Data Interpretation — Table & Bar/Pie Chart 🔴

## 📌 What is Data Interpretation?
DI involves analyzing data presented in tables, charts, and graphs to answer questions. No new formulas — it uses percentage, ratio, average concepts on given data.

---

## 📝 Types of DI

### Table-based DI
- Data in rows and columns
- Read values carefully — units matter!
- Common questions: percentage change, ratio, average, max/min

### Bar Graph
- Heights represent values
- Compare bars to find differences, ratios
- Stacked bars: each segment = component

### Pie Chart
- Total = 360° = 100%
- Each slice: Value = (Angle/360) × Total OR (Percentage/100) × Total
- Central angle = (Value/Total) × 360°

### Line Graph
- Shows trends over time
- Slope = rate of change
- Steeper = faster change

---

## 📝 Key Calculation Skills

### Percentage Change
```
% Change = [(New - Old) / Old] × 100
```

### Percentage of Total
```
% = (Part / Total) × 100
```

### Ratio
```
Ratio = A : B (simplify by dividing by HCF)
```

### Average
```
Average = Sum of values / Number of values
```

## ⚡ Tricks & Shortcuts

> 💡 **Trick 1 – Approximation**: In DI, approximate! 298 ≈ 300, 197 ≈ 200. Speed matters more than exact accuracy.

> 💡 **Trick 2 – Percentage shortcuts**: 10% = divide by 10. 5% = half of 10%. 1% = divide by 100. Build up from these.

> 💡 **Trick 3 – Pie Chart**: 1% = 3.6°. 10% = 36°. 25% = 90°. Use these anchors.

> 💡 **Trick 4 – Elimination**: In MCQs, estimate and eliminate clearly wrong options before precise calculation.

---

## 📖 Example Approach

### For a Table Question:
1. Read the question carefully — what exactly is asked?
2. Locate the relevant row/column
3. Apply the formula (%, ratio, average, etc.)
4. Approximate if options are far apart

### For Pie Chart:
1. Note the total value
2. Calculate each sector's value = (angle/360) × total
3. Apply comparisons as asked
