# Bar & Pie Chart DI 🔴

## 📝 Bar Chart Reading
- Y-axis = values, X-axis = categories/years
- Height of bar = data value
- Grouped bars: compare different series side by side
- Stacked bars: components sum to total

## 📝 Pie Chart Formulas
```
Value of sector = (Central angle / 360°) × Total
Central angle = (Value / Total) × 360°
Percentage = (Central angle / 360°) × 100
```

### Angle ↔ Percentage Reference
| Percentage | Angle |
|------------|-------|
| 10% | 36° |
| 20% | 72° |
| 25% | 90° |
| 33.33% | 120° |
| 50% | 180° |

## 📖 Sample DI Problem

**Given**: Pie chart shows expenses — Food: 30%, Rent: 25%, Transport: 15%, Others: 30%. Total income = ₹50000.

**Q1**: Amount spent on Food?  
**Ans**: 30% of 50000 = **₹15000**

**Q2**: Angle for Transport?  
**Ans**: 15% × 360° = **54°**

**Q3**: Ratio of Rent to Transport?  
**Ans**: 25:15 = **5:3**

---

## 🎯 Key Strategy
1. Always note the total first
2. Convert angle ↔ percentage ↔ value as needed
3. For comparison questions, compute both values then compare
4. Approximate aggressively when options are spread out
