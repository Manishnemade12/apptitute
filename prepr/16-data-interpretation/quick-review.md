# ⚡ Quick Review: Data Interpretation 🔴

---

## Key Formulas Used in DI
```
% Change = (New-Old)/Old × 100
% of Total = Part/Total × 100
Average = Sum/Count
Ratio = A:B (simplify)
```

## Pie Chart
- Value = (Angle/360) × Total | Angle = (Value/Total) × 360
- 10%=36° | 25%=90° | 50%=180°

## Strategy
1. Read question first → identify what's asked
2. Locate data in table/chart
3. **Approximate** — don't waste time on exact calculations
4. Eliminate wrong options fast
